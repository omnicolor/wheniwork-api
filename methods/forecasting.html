Get Projected Sales

Example Request

curl https://api.wheniwork.com/2/forecasting/projected \
 -H "W-Token: ilovemyboss"

<?php
$wiw = new Wheniwork("ilovemyboss");
$result = $wiw->get("forecasting/projected");
?>

HTTP Request

GET https://api.wheniwork.com/2/forecasting/projected

Parameters

Key Description
start datetime
(Optional) The start of the time interval to get projected sales for.
end datetime
(Optional) The end of the time interval to get projected sales for.
location_id integer
(Optional) The ID of the location to get projected sales for.
